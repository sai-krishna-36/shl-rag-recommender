SHL Assessment Recommendation System

This application recommends relevant SHL assessments based on natural language hiring or role-related queries.

The system uses semantic similarity to match job requirements with appropriate technical and behavioral assessments from the SHL catalog.

How to use this application

  • Open the interactive API documentation at /docs
  • Use the POST /recommend endpoint
  • Provide a hiring-related query as input

Example query:

Hiring a Java developer with good communication skills

This project is designed as an API-first service and can be integrated into any frontend or internal HR tool.
